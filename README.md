# Proxia IA

Site vitrine + CRM pour agence digitale spécialisée commerçants locaux.

**Stack :** React 19 + Vite + Tailwind CSS + Framer Motion + React Router v7
**Live :** https://proxia-smoky.vercel.app
**Dashboard PIN :** 2611

## Variables d'environnement

Copier `.env.example` en `.env` et renseigner vos clés :

```
VITE_GOOGLE_MAPS_KEY=
VITE_GEMINI_KEY=
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
```

## Développement

```bash
npm install
npm run dev
```
